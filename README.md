# 🎓 EduManage | Enterprise Student Management System

[![GitHub License](https://img.shields.io/github/license/poojaspy9730/internship-project-SDM-)](https://github.com/poojaspy9730/internship-project-SDM-/blob/main/LICENSE)
[![Node.js Version](https://img.shields.io/badge/node-%3E%3D%2014.0.0-brightgreen)](https://nodejs.org/)

**EduManage** is a modern, high-performance Student Management System designed for academic excellence. It features a sleek glassmorphic dashboard, real-time student tracking, and an integrated AI chatbot for a premium administrative experience.

## ✨ Key Features

- **🚀 Glassmorphism Dashboard:** A stunning, modern UI built for clarity and speed.
- **📁 Full CRUD Operations:** Effortlessly Create, Read, Update, and Delete student records.
- **🤖 Smart AI Chatbot:** An interactive assistant to help you navigate and manage data.
- **🔍 Dynamic Search & Filter:** Instant results as you type.
- **🛡️ SQLite Security:** Fast and reliable relational database storage.
- **📱 Responsive Design:** Fully optimized for mobile, tablet, and desktop views.
